# Procedural-Programing-Fundamentals

This repository contains **6 months of hands-on practice in C**, covering key topics in procedural programming.

## What’s Inside

- **Data Structures:** Arrays, stacks, single and doubly linked lists, points in 2D space.  
- **Memory Management:** Dynamic allocation, cleanup, and safe operations.  
- **File Handling:** Reading and writing arrays and generic data types to binary files.  
- **Modular Programming:** Multi-file projects, static and dynamic libraries.  
- **Unit Testing & TDD:** Writing tests with `assert.h` and ensuring correctness.  
- **Sorting Algorithms:** Bubble sort, insertion sort, selection sort, merge sort, quicksort.  
- **Algorithm Analysis:** Counting operations, measuring execution time, and comparing efficiency.

## Skills Practiced

- Writing modular and reusable C code.  
- Implementing and testing core data structures from scratch.  
- Understanding algorithmic complexity and efficiency.  
- Working with files, binary data, and libraries.  
- Applying test-driven development principles.

**Summary:**  
Through these projects, I strengthened my understanding of **procedural programming, C language fundamentals, and algorithmic thinking**, creating practical solutions to classical computer science problems.
